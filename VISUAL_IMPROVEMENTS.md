### 5. 主题切换支持 ✅
- **实时检测**：每次更新时间时检查系统主题
- **动态适配**：背景色和文字色随主题自动切换
- **阴影调整**：阴影透明度随主题变化（深色主题更明显）

```python
# 阴影效果设置
def _setup_shadow(self):
    effect = QGraphicsDropShadowEffect()
    effect.setBlurRadius(15)
    effect.setOffset(0, 3)
    effect.setColor(QColor(74, 144, 226, 128))
    self.setGraphicsEffect(effect)

# 主题自适应
def _update_theme(self):
    palette = app.palette()
    bg_color = palette.color(QPalette.Window)
    is_dark = bg_color.lightness() <= 128
    
    if is_dark:
        bg_color_str = "rgba(28, 28, 28, 0.95)"
        text_color_str = "#87CEFA"
        shadow_opacity = 179
    else:
        bg_color_str = "rgba(255, 255, 255, 0.95)"
        text_color_str = "#4A90E2"
        shadow_opacity = 38
```

## ⚠️ 潜在优化点

### 1. 性能优化
- **阴影渲染**：`QGraphicsDropShadowEffect` 可能有轻微性能开销
- **频繁重绘**：每秒检测主题可能没必要
  - 建议：仅在启动时或用户操作时检测主题

### 2. 主题切换监听
- **当前实现**：每秒检测一次主题（在 `update_time()` 中）
- **优化建议**：使用 `QApplication.paletteChanged` 信号监听主题切换

### 3. 拖动性能
- **阴影影响**：拖动时阴影可能导致轻微卡顿
- **优化方案**：拖动时临时禁用阴影
